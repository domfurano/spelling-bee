# spelling-bee

A NYT Spelling Bee clone built with TypeScript.

## 🎮 Live Demo

Play the game at: **https://domfurano.github.io/spelling-bee/**

## 🚀 Getting Started

```bash
npm install
npm start
```

## 🏗️ Build

```bash
npm run build
```

The production build is output to the `dist/` directory.

## 🧪 Tests

```bash
npm test
```
